Projects

Real-Time Object Detection for Construction Safety

Walbridge & U of M MDP

A collaborative industry-sponsored project aiming to enhance construction site safety using AI-driven hazard detection. The team and I are developing a real-time object detection model using a CNN and YOLOv8, which was trained on 20GB+ of real construction site video data. The model detects and classifies site hazards with a current accuracy of 87%, with ongoing work to improve performance.

Python PyTorch TensorFlow Keras YOLOv8

Instagram Clone

Full Stack Web Application

A full-featured clone of Instagram built from scratch to replicate core functionality like user authentication, post uploads, likes, comments, and infinite scrolling. Medlytics Program

MIT Beaver Works

An intensive summer program focused on machine learning in medicine. I developed several ML models including random forests, SVMs, and neural networks for predicting conditions such as hypothyroidism and cancer. My capstone project involved building a convolutional neural network to classify retinal scans and detect early signs of eye conditions such as glaucoma or cataracts.

Python TensorFlow Keras NumPy pandas
